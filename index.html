To connect real Instagram accounts, you need a free Meta Developer App. Takes about 10 minutes.
1Go to developers.facebook.com → Create App → Choose Business type
2Add the Instagram Basic Display product to your app
3Under Instagram Basic Display → Settings, add your page URL as a Valid OAuth Redirect URI
4Copy your Instagram App ID and paste it into this file at: CONFIG.INSTAGRAM_APP_ID
5To exchange the auth code for a token (needed for real data), you'll need a small backend endpoint. Ask Claude to build you a simple Node.js or Python handler.
